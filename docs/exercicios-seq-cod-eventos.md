3. O programa abaixo está exibindo sempre duas mensagens
    - Valor 1 é maior
    - Valor 2 é maior

    Corrija o programa para que ele teste:
    
    ```
     se valor 1 é maior que valor 2:

Senão se ataque menor que defesa:
    escrever "Bem vindo ladrão!"
```
